Help Information


Site Page:


The main site page is split into 12 subsections, the first button available is the home button this will take you back to the welcome page of the website at any time should you need to return there. The other 11 buttons split the page into console subsections clicking any of these buttons will load up the corresponding console and all the available reviewed games of that console, not all reviews where done on said console but they are available on the console so will be displayed, this loaded tab can then be separately scrolled through at leisure to locate a review.


Search Function:


The search function will always be displayed on the left hand side of any page, from this search function the user can generate there own filtered results for any console. There are two mandatory fields in the search function Platform and Age these two fields must be filled to generate a result all other criteria such as rating, co-operative play and online availability are optional and are there to narrow your search down to a very specific level
